# 🚀 Blogging Application

> A modern blogging platform backend built with Express, TypeScript, Prisma, and PostgreSQL.

## ✨ Features

- User Authentication
- JWT Authorization
- Role-Based Access Control
- Blog Management
- Comment System
- Prisma ORM
- Pagination & Filtering

## 🛠 Tech Stack

- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT
- bcrypt
